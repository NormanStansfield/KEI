# KEI

Keyword Editor for IPTC data.

## Features 
Simple read / write title and keywords of IPTC data with GUI.
This is one of frond-end of ExifTool.

## Support Tags

- ObjectName
- Keywords

charset will overwrite to UTF8.

## Support Image File Format

- JPEG
